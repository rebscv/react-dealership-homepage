# React Dealership Homepage

This project is a dealership-style homepage built using **React** and **Vite**.

The goal of this project is to demonstrate **component-based architecture**, **dynamic rendering**, and the progressive development of a frontend project from structure to a fully styled UI.

---

## Tech Stack

- React
- Vite
- JavaScript (ES6)
- CSS

---

## Project Structure

The project is organised around reusable components.

src
├ components
│ ├ QuickLinks
│ ├ QuickLinkCard
│ └ (other components)
│
├ pages
│ └ HomePage.jsx
│
├ App.jsx
└ main.jsx


---

## Development Stages

This project is being built in three stages to demonstrate the development process.

### Version 1 – Component Architecture

Focus on React fundamentals:

- Reusable components
- Props
- Dynamic rendering with `map()`
- Basic page structure

Sections include:

- Header
- Hero
- QuickLinks
- Featured Stock
- Welcome
- Footer

Minimal styling is used at this stage.

---

### Version 2 – Layout & Styling

Introduce layout and visual design:

- CSS Grid / Flexbox
- Spacing and typography
- Responsive layout

---

### Version 3 – Full UI Implementation

Final polished version including:

- Images and branding
- Icons
- Hover interactions
- Fully responsive design
